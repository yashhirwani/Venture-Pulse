## рџ“Њ Project Overview

**Venture Pulse** is a machine learning powered web application designed to predict whether a startup is likely to succeed or fail based on key business and financial parameters.

The application integrates:

* A **web-based frontend** for user input
* A **Python backend** for processing and prediction
* A **machine learning model** trained on startup data

This project demonstrates the practical integration of **Machine Learning with full-stack web development**.

---

## рџЋЇ Problem Statement

Investors and entrepreneurs often struggle to evaluate the potential success of startups due to multiple influencing factors such as funding, market sector, and team structure.

**Venture Pulse** attempts to solve this problem by using machine learning models to analyze historical startup data and predict startup success.
